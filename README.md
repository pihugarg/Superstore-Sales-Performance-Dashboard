# 📊 Superstore Sales Performance Dashboard

An interactive **Power BI dashboard** built using the **Sample Superstore dataset** to analyze sales, profit, customer orders, and regional performance. The dashboard provides business insights through KPIs, interactive visualizations, slicers, and conditional formatting to support data-driven decision-making.

---

## 📌 Project Overview

The Superstore Sales Performance Dashboard is designed to help business users monitor overall performance, identify sales trends, evaluate product profitability, and compare regional performance. The dashboard consolidates key business metrics into a single-page interactive report, making it easy to explore data and uncover actionable insights.

---

## 🎯 Objectives

- Monitor key business KPIs.
- Analyze monthly and yearly sales performance.
- Compare sales and profit across product categories.
- Identify high-performing and low-performing sub-categories.
- Evaluate regional sales and profitability.
- Enable interactive analysis using slicers.

## 🎨 Conditional Formatting

Conditional formatting has been applied to the **Profit by Sub-Category** bar chart to improve data interpretation.

- 🟢 **Green Bars** → Positive Profit
- 🔴 **Red Bars** → Negative Profit (Loss)

This allows users to quickly identify profitable and loss-making product sub-categories without manually comparing values.
